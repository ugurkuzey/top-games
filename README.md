# Play Store Top 10 Games

Get top 10 games from google play store and get detailed information about games using .NET5 API project.

## Prerequirements

* Visual Studio 2019
* .NET 5 SDK

## How To Run

* Open solution in Visual Studio 2019
* Run the application.

## Tech
- .NET 5
- SQLite
